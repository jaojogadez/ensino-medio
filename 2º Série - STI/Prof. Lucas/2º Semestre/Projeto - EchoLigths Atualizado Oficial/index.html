Sobre o Projeto:

Sobre o Projeto:

A EchoLights é uma inovadora árvore de Natal interativa, cercada por LEDs cujas cores podem ser personalizadas. O sistema utiliza um sensor de som programado para detectar palmas, permitindo controlar a árvore de maneira dinâmica. De acordo com a quantidade de palmas, os LEDs executam diferentes comandos.

Abaixo mostramos a lista dos componentes para este projeto!

Tutorial da montagem:

Segue abaixo o passo a passo para montagem do projeto.

Sobre o sensor de som:

O sensor KY-038 é fundamental para o funcionamento do projeto.

Logo, precisamos conhecer mais sobre ele.

  • A0 Saída analógica.
  • G⠀ Alimentação negativa.
  • +⠀ Alimentação positiva.
  • D0 Saída digital.

1 - Monte o circuito KY-038

Veja ao lado o circuito para ligar o sensor de som. Apenas para testes indivuais.

O LED Vermelho representa a fita LED.

Vale salientar que o sensor KY-037 necessita de ajustes de sensibilidade.

Sobre a fita LED:

A fita LED RGB endereçável WS2812B, é outra peça fundamental do nosso sistema, é também a fita mais moderna do mercado atualmente.

Pontos Importantes:

  • Como a fita LED é pequena não é necessário fonte 5v externa.
  • Cada LED pode ser controlado individualmente.
  • Logo abaixo mostraremos o código.

2 - Monte o circuito da Fita LED

Veja ao lado o circuito que usamos para testar os LEDs.

Agora que sabemos das peças que iremos usar, é hora de juntar os circuitos e criar o Echo Lights!

Funcionamento:

Copie o código abaixo e cole no Arduino IDE. Não se esqueça de incluir a biblioteca Adafruit_NeoPixel.h.

                // Adicao da biblioteca de controle da fita de LED
                #include <Adafruit_NeoPixel.h>

                // Variavel que armazena o numero de LEDs que serao controlados
                const int NUMERO_LEDS = 60; // Padrao para 1 metro
                // Caso queira controlar mais do que 1 metro, altere o valor acima com o numero de LEDs correspondente a metragem

                // Variavel que armazena o pino de controle da fita de LEDs
                const int PINO_FITA_LEDS = 2;

                // Objeto de controle da fita de LEDs
                Adafruit_NeoPixel fita_LED(NUMERO_LEDS, PINO_FITA_LEDS);

                int led = 8;//Atribui o pino 8 a variável led
                int ky = A0;//Atribui o pino 9 - conectado ao sensor, a variável ky
                int valorky = 0;//Declara a variável valorky do tipo inteiro
                bool statusLed = false;//Declara a variável statusLed do tipo booleano


                void setup() {

                  pinMode (led, OUTPUT);//Define led (porta digital 8) como saída
                  pinMode (ky, INPUT);//Define ky (porta analógico 9) como saída
                  Serial.begin(9600);//Inicialização da comunicação serial, com velocidade de comunicação de 9600

                  fita_LED.begin(); // Inicializacao da fita de LEDs

                  // Inicia com os LEDs da fita apagados
                  fita_LED.clear();
                  fita_LED.show();
                  delay(100);

                  // Realiza um aparecimento gradual acendendo da cor vermelha em todos os LEDs
                  for (int i = 0; i < 256; i ++) { // Varia a cor do seu minimo (0) ate o seu maximo (255)
                    for (int j = 0; j < NUMERO_LEDS; j++) {
                      fita_LED.setPixelColor(j, i, 0, 0); // Atualiza a cor de todos os LEDs da fita com o ultimo valor
                    }
                    fita_LED.show(); // Exibe o ultimo controle salvo nos LEDs
                    delay(5);
                  }

                  delay(100); // Aguarda 500 ms para nova cor

                  // Apaga os LEDs para nova cor
                  fita_LED.clear();
                  fita_LED.show();

                  // Realiza um aparecimento gradual acendendo da cor verde em todos os LEDs
                  for (int i = 0; i < 256; i ++) { // Varia a cor do seu minimo (0) ate o seu maximo (255)
                    for (int j = 0; j < NUMERO_LEDS; j++) {
                      fita_LED.setPixelColor(j, 0, i, 0); // Atualiza a cor de todos os LEDs da fita com o ultimo valor
                    }
                    fita_LED.show(); // Exibe o ultimo controle salvo nos LEDs
                    delay(5);
                  }

                  delay(100); // Aguarda 500 ms para nova cor

                  // Apaga os LEDs para nova cor
                  fita_LED.clear();
                  fita_LED.show();

                  // Realiza um aparecimento gradual acendendo da cor azul em todos os LEDs
                  for (int i = 0; i < 256; i ++) { // Varia a cor do seu minimo (0) ate o seu maximo (255)
                    for (int j = 0; j < NUMERO_LEDS; j++) {
                      fita_LED.setPixelColor(j, 0, 0, i); // Atualiza a cor de todos os LEDs da fita com o ultimo valor
                    }
                    fita_LED.show(); // Exibe o ultimo controle salvo nos LEDs
                    delay(5);
                  }
                  delay(100); // Aguarda 500 ms para nova cor

                  // Apaga os LEDs para nova cor
                  fita_LED.clear();
                  fita_LED.show();

                }

                
                  void loop() {
                    valorky = analogRead(ky);//Lê o valor do sensor e armazena na variável valorky
                    Serial.print("\nSaida A0: "); // Imprime no serial monitor "\nSaida A0: "
                    Serial.print(valorky); // Imprime no serial monitor o valor que a variável valor_A0 recebeu
                    delay(20);//Intervalo de 20 milissegundos
                      if (valorky > 100 && statusLed == false) { //Se valorky for igual a 1 (HIGH) e statusLed for igual a false:
                          delay(100);
                          // Gera valores RGB aleatorios
                          randomSeed(analogRead(A0));
                          int vermelho = random(0, 255);
                          int verde = random(0, 255);
                          int azul = random(0, 255);
                  
                          // Acende os LEDs com os valores RGB gerados
                          for (int l = 0; l < NUMERO_LEDS; l++) {
                            fita_LED.setPixelColor(l, vermelho, verde, azul); // Atualiza a cor dos LEDs com os valores coletados
                          }
                          fita_LED.show(); // Exibe o ultimo controle salvo nos LEDs
                          delay(500); // Espera para nova cor
                            statusLed = true;//Coloca statusLed como true
                          }
                  
                      else {//Senão:
                  
                          delay(100);
                          for (int l = 0; l < NUMERO_LEDS; l++) {
                            fita_LED.setPixelColor(0, 0, 0, 0); // Atualiza a cor dos LEDs com os valores coletados
                          }
                          fita_LED.show(); // Exibe o ultimo controle salvo nos LEDs
                          delay(500); // Espera para nova cor
                            statusLed = false;//Coloca statusLed como true
                          }
                  }

Explicação do Código:

Em resumo, com o código rodando, o sensor KY-038 será capaz de "ouvir" o som de uma palma. Quando o volume captado pelo sensor ultrapassar o valor de 100, ele enviará um sinal para acender a fita LED. Ao receber outro sinal com o mesmo nível de intensidade, os LEDs serão apagados.
